## 42sh

Features:

Edition de ligne:

    -edition simple en multiligne
    -Selection            : ALT + Fleche
    -Copier/Coller        : ALT + C ALT + V
    -Couper               : ALT + X
    -Debut de ligne       : HOME
    -Fin de ligne         : END
    -Deplacement par mots : SHIFT + Fleche G/D
    -Deplacement par ligne: SHIFT + Fleche H/B

Built In:
    -Echo
    -Exit
    -Setenv
    -Unsetenv
    -Export
    -unset
    -Local
